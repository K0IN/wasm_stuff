Call async Javascript from C

Using C++ coroutines to call async js

This Example uses the browser fetch api from C++
Javascript calls the runtime as a sort of event loop 

resumable foo(int x)
{
    while (true)
    {
        int f = do_fetch(x);
        if (f)
        {
            co_return f;
        }
        co_await std::experimental::coroutines::suspend_always();
    }
}
